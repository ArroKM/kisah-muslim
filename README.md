<h3>Installing</h3>

```java
$ apt update && apt upgrade
$ termux-setup-storage;mkdir $HOME/storage/shared/KISAH-MUSLIM
$ apt install git
$ apt install python
$ pip install requests
$ pip install bs4
$ git clone https://github.com/ArroKM/kisah-muslim
$ cd kisah-muslim
$ python3 kisah.py
```
